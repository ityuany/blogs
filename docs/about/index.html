    这里记录了一些个人常用的工具集。

在线工具

  • carbon 制作漂亮的代码图片。
  • can i use 查询浏览器的特性支持情况
  • Package Diff 比较 npm 包两个版本直接的区别

mac os 常用工具

  • kap 好用的截屏制作 GIF 动图软件、可以手动调节帧率、大小，颜值高
  • surge + tempest(原 rixCloud) 科学上网、xx 时期，xxx 定。
  • ColorSnapper2 取色小工具，轻巧实用。
  • fork 高颜值的 Git GUI、关键不卡、也不用注册账号，比 source-tree 好用。
  • coreshell 一个高颜值的 SSH 管理器
  • paw 类似于 PostMan、高颜值。不过功能没有 PostMan 强大，但是基础功能还是实用的。
  • motrix 下载工具
  • charles 小花瓶、http 代理。抓包调试好用。
  • CatchMouse 这是一个可以设置快捷键快速的在多个副屏之间来回切换的软件、遗憾的是作者不维护了，不过暂时找不到好的替代品。
  • Rocket 快速选取 Emoji 表情.
  • runcat 小猫跑跑、显示 CPU 负荷量。
  • frp 一个高性能反向代理服务器，可以帮助你将本地服务器暴露在 NAT 或者防火墙之后。
  • gridea 类似于 GUI 版本的 hexo ，我自己不用，不过有点意思。
  • DockStation 基于 eletron 的 docker gui

react 生态

  • swr 用于远程数据获取,里面封装了大量的骚操作，建议直接取看 demo。
  • react-use hooks 工具集。
  • ahooks 阿里巴巴开源的 hooks 工具集。

好用的 JS 库

  • superagent 轻量级的渐进式 ajax API，在对许多现有的请求 API 感到沮丧之后，它具有灵活性、可读性和较低的学习曲线。
  • xterm.js 在网页上的终端工具。
  • path-to-regexp 路由的匹配库，vue、react 的 router 基本上都是基于它实现的。

Node 工具库

  • robotjs 可以控制系统鼠标翩翩起舞（感觉目前功能还比较弱鸡，持续关注。）
  • commander.js 命令行工具
  • micromatch 基于字符串、复杂的路径规则匹配
  • dockerode 封装了一些 docker 的操作 API
  • puppeteer 爬虫，爬爬爬。
  • shelljs Node.js 的可移植 Unix Shell 命令
  • nodegit Git Node 工具
  • chokidar 一个高效的 node.js fs.watch/fs.watchFile/FSEvents 包装器
  • bili 基于 rollup，简化配置。
  • fx 交互式 json 查看。
  • qrcode-terminal 二维码生成
  • concurrently 在 npm scripts 里并行执行命令。
  • jscpd 代码质量分析，主要是通过分析代码重复率来判定代码质量，我自己没有用过

Markdown

  • remark Markdown 语法解析器。
  • markdown-it Markdown 转 HTML

Java 好用的好东西

  • mapstruct 更好的通过编译层来解决 copy properties。
